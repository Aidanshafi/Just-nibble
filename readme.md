## Just Nibble

Just Nibble is a simple Python program using Tkinter that recommends meals based on user input, such as hunger, calorie intake, diet, budget, and allergies

## How to use

1. Create a new folder in VSCode and name it "Just Nibble".
2. Create a new file with the name "main.py".
3. Copy the code by going into the repository, which you can find in the word file as a URL.
4. Open the "user_inputs" file and copy the complete code from there and paste it into "main.py".
5. Make sure you are using Python 3.14.0, which is the version the code was created in.
6. Create a new file and name it "meals.json" in the "Just Nibble" folder.
7. Go into the repository again and find the "meal.json" file.
8. Open the "meal.json" file and copy the complete contents from there and paste it into "meals.json".
9. Select the "main.py" and run the program.
10. Put in your desired inputs. 
11. Congrats! You did it!
